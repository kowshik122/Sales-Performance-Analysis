# Sales Performance Analysis

## Project Overview
This project analyzes retail sales data to identify trends, profit drivers, and loss-making categories using Python and data visualization techniques.

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Excel
- Power BI (optional)
- GitHub

## Key Analysis Performed
- Data cleaning and preprocessing
- Monthly sales trend analysis
- Category-wise profit analysis
- Business insights generation

## Key Insights
- Identified peak sales months
- Found most and least profitable product categories
- Provided data-driven insights for decision making
